## What This Is

A hand-built portfolio and technical blog. No frameworks, no templates — pure HTML, CSS, and JavaScript with Jekyll for the blog pipeline. Deployed on GitHub Pages.

---

## Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Blog engine:** Jekyll
- **Hosting:** GitHub Pages
- **Fonts:** Syne, Space Mono, Instrument Serif (Google Fonts)

---

## File Structure

```
laughing-succotash/
├── _posts/                  ← Jekyll blog post markdown files
├── css/
│   └── style.css            ← All styles (design tokens, layout, responsive)
├── js/
│   ├── main.js              ← Cursor, scroll reveal, hamburger menu
│   └── layout.js            ← Shared nav/footer injector
├── pages/
│   ├── post-strict-equality.html
│   └── post-closures.html
├── index.html               ← Home page
├── about.html               ← About page
├── blog.html                ← Blog listing
├── works.html               ← Projects page
├── _config.yml              ← Jekyll config
└── README.md
```

---

## Running Locally

```bash
# Install Jekyll if you haven't
gem install jekyll bundler

# Serve locally
bundle exec jekyll serve

# Open http://localhost:4000
```

Or just open `index.html` directly in a browser — everything works without Jekyll except the `_posts` rendering.
