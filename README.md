## ✦ About

This is my personal developer portfolio and technical blog, built from scratch without templates or UI kits.[page:1]  
The site reflects an industrial, **intentional** design approach: clear hierarchy, minimal noise, and hand‑written HTML, CSS, and vanilla JavaScript.[page:1]  
Blog content is powered by Jekyll, and the site is deployed via GitHub Pages.[page:1]

Key goals:

- Present projects and skills in a clean, readable way.
- Host long‑form technical posts (JavaScript, CSS, Git, etc.).[page:1]
- Maintain a portfolio that is fully under my control: no drag‑and‑drop builders, no boilerplate themes.[page:1]

## Running the Project

### Option 1 – Simple static view

If you only want to see the portfolio pages (without Jekyll blog processing):

```bash
git clone https://github.com/SniperRavan/sniperravan-portfolio.git
cd sniperravan-portfolio
# Then open index.html in your browser (double‑click or use your editor's live server)
```

This will load `index.html`, `about.html`, `blog.html`, and `works.html` as plain static pages.[page:1]

### Option 2 – Full Jekyll blog

To run the blog with Jekyll (recommended for local development): [page:1]

```bash
gem install jekyll bundler   # if not already installed
bundle exec jekyll serve
```

Then open:

```text
http://localhost:4000
```

This will render posts from `_posts/` and apply the Jekyll pipeline. [page:1]
